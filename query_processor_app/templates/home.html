The Durham City Info Lookup Tool, aka StreetWise is being created by Code for Durham as a resource for all Durham residents.  The user can enter a Durham address and be returned various useful data items. The site is being designed to be helpful to everyone, especially those who are not web savvy or don't have readily available internet access.  All Durham residents are welcome to use this tool however they find it helpful.

Please note the site is currently under development.  It may be slow and information may be incorrect.

Thank you for visiting the site.  For more information, visit CodeForDurham.com.
